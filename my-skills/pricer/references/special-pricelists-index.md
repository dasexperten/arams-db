## How to Add a New Special Price List

1. Create file: `pricelist-[id].md` (e.g., `pricelist-special-torwey.md`)
2. Use the standard template from SKILL.md
3. Add one row to the table above
4. Add one row to the Special Price List Registry in SKILL.md

### Template reminder:

```markdown
# [Buyer Name] — Special Prices
**Price List ID:** `SPECIAL-[NAME]`
**Currency:** [USD / RUB / CNY]
**Valid from:** [date]
**Applies to:** [Seller entity] → [Buyer legal name]
**Authorized by:** [reference to agreement or person]
**Notes:** [conditions, expiry, minimum order, etc.]

---

|-----|-------------|---------------|---------------------|------------|

```
