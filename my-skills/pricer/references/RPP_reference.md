# PARTNER RETAIL PRICER (RPP)
**Reference file** — loaded by Pricer skill on demand only.

## MODULE: PARTNER RETAIL PRICER (RPP)
**RU:** Ритейл-ценообразование для партнёров

> ⚠️ Данные по пошлинам, НДС, фрахту зафиксированы для **России** как константы.
> Для всех других стран — RPP делает gate-запросы к `legalizer` и `logist` перед расчётом.

- `RPP`

---

### Что делает этот модуль
Рассчитывает цену входа партнёра (дистрибьютора) в розничную сеть на основе цен DEI и желаемой маржи партнёра.

**Логика: обратный отсчёт от маржи.**
Партнёр задаёт желаемую маржу → модуль считает цену входа в сеть с полной раскладкой.


Никогда не смешивать два прайса в одном расчёте.

### ШАГ 3 — СБОР ДАННЫХ ПО ПОШЛИНАМ И НДС

#### 🇷🇺 Россия — константы (gate не нужен)

| Параметр | Значение | Источник |
|---|---|---|
| Пошлина — пасты (DE2xx) | 6.5% от FOB | Таможенный тариф РФ |
| Пошлина — щётки, нити, ополаскиватели | 15% от FOB | Таможенный тариф РФ |
| НДС ввозной | 20% от (FOB + фрахт + пошлина) | НК РФ — возмещается |
| НДС исходящий (продажа в сеть) | 20% | НК РФ — возмещается сетью |

#### 🌍 Другие страны — gate-запрос к legalizer

```
[[GATE: legalizer-customs]]
Country: [страна партнёра]
Product categories: [Паста / Щётка / Нить / Ополаскиватель]
```

Legalizer возвращает: пошлину по категориям + ставку НДС + признак возмещаемости.
После получения данных: `↩️ Legalizer gate complete — resuming RPP.`

### ШАГ 6 — ФОРМУЛА РАСЧЁТА

```
Таможенная стоимость       = (FOB + Фрахт) × курс
Таможенный НДС             = Таможенная стоимость × НДС%  [возмещается — в себестоимость НЕ входит]
База пошлины               = Таможенная стоимость + Таможенный НДС
Пошлина                    = База пошлины × пошлина%
Landing Cost (себестоимость) = Таможенная стоимость + Пошлина  [НДС не включён]
Цена в сеть без НДС        = Landing Cost ÷ (1 − маржа%)
Цена в сеть с НДС          = Цена без НДС × (1 + НДС исходящий%)
```

**Маржа (PM) считается от цены в сеть без НДС.**
**НДС ввозной возмещается партнёром — в Landing Cost не входит.**

```
* Таможенный НДС уплачивается на границе и полностью возмещается.
```

Если запрошен один SKU — детальная разбивка построчно.
Если запрошена полная таблица — табличный формат по всем SKU активного прайса.
